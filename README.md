# SuperMemory-VQA Project Page

Standalone static website for the SuperMemory-VQA project.

## Local Preview

Open `index.html` directly, or serve the folder with any static file server.

## Contents

- `index.html`: project page content.
- `static/css/supermemory.css`: project-specific styling using the manuscript color palette and ByteSans fonts.
- `static/js/supermemory.js`: lightweight reveal and count-up animations.
- `static/images/supermemory/`: converted manuscript figures and selected qualitative examples.
- `static/images/logos/`: OSU and Meta logos.
- `static/pdfs/supermemory-vqa.pdf`: current paper PDF.

The page was initialized from a standard academic project-page template and then customized for SuperMemory-VQA.
