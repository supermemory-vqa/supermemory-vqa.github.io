# SuperMemory-VQA Project Page

Standalone static website for the SuperMemory-VQA project.

## Local Preview

Open `index.html` directly, or serve the folder with any static file server.

## Contents

- `index.html`: project page content.
- `static/css/supermemory.css`: styling. The design system is ported from the
  [MMDeepResearch-Bench (MMDR-Bench)](https://github.com/MMDeepResearch-Bench/MMDeepResearch-bench.github.io)
  template: a clean light theme with the Inter typeface, card-based layout, and band separators, using the OSU
  scarlet accent.
- `static/js/supermemory.js`: reveal/count-up animations plus the interactive (filter / search / sortable) leaderboard.
- `static/images/supermemory/`: converted manuscript figures and qualitative examples.

The layout follows the MMDR-Bench template (sticky nav, gradient hero with author block, quickstart cards, figure
cards, and an interactive leaderboard) with SuperMemory-VQA content.
